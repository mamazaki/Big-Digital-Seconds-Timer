# Big Digital Seconds Timer / นาฬิกานับถอยหลังดิจิทัลขนาดใหญ่

A simple, single-file HTML countdown timer designed for presentations, events, or personal use.  
นาฬิกานับถอยหลังแบบไฟล์ HTML เดียวที่ใช้งานง่าย ออกแบบมาสำหรับการนำเสนอ กิจกรรม หรือการใช้งานส่วนตัว

## Features / คุณสมบัติ

- **Large Digital Display**: Easy to read from a distance.  
  **จอแสดงผลขนาดใหญ่**: มองเห็นได้ชัดเจนจากระยะไกล
- **Custom Timer**: Set any specific duration in seconds.  
  **กำหนดเวลาเองได้**: ระบุระยะเวลาเป็นวินาทีตามต้องการ
- **Quick Presets**: Ready-to-use buttons for 93 seconds and 60 seconds.  
  **ปุ่มทางลัด**: มีปุ่มสำหรับเริ่มจับเวลา 93 วินาที และ 60 วินาทีทันที
- **Visual Warning**: The timer turns **RED** when less than 10 seconds remain.  
  **แจ้งเตือนด้วยภาพ**: ตัวเลขจะเปลี่ยนเป็น **สีแดง** เมื่อเหลือเวลาน้อยกว่า 10 วินาที
- **Full Screen Mode**: Toggle full screen for an immersive experience.  
  **โหมดเต็มหน้าจอ**: สามารถกดปุ่มเพื่อขยายเต็มหน้าจอได้
- **Responsive Design**: Adjusts font sizes automatically for smaller screens.  
  **รองรับทุกหน้าจอ**: ปรับขนาดตัวอักษรอัตโนมัติสำหรับการแสดงผลบนมือถือหรือแท็บเล็ต
- **Color Customization**: Pick any color or use quick samples (Green, Cyan, Magenta, Yellow, Orange, White).  
  **ปรับแต่งสีได้**: เลือกสีตัวเลขได้อิสระ หรือใช้สีตัวอย่าง (เขียว, ฟ้า, ชมพู, เหลือง, ส้ม, ขาว)
- **Thai Numerals**: Toggle between Hindu-Arabic (123) and Thai (๑๒๓) numerals.  
  **รองรับเลขไทย**: สลับการแสดงผลระหว่างเลขอารบิกและเลขไทยได้ง่ายๆ

## How to Use / วิธีการใช้งาน

1. Download or copy the `countdown.html` file.  
   ดาวน์โหลดหรือคัดลอกไฟล์ `countdown.html`
2. Open the file in any modern web browser (Chrome, Edge, Firefox, etc.).  
   เปิดไฟล์ด้วยเว็บเบราว์เซอร์ (เช่น Chrome, Edge, Firefox)
3. Enter seconds in the input box and click **"เริ่ม (กำหนดเอง)"**, or use the preset buttons.  
   กรอกตัวเลขวินาทีในช่องแล้วกด **"เริ่ม (กำหนดเอง)"** หรือใช้ปุ่มตัวเลือกเวลาที่มีให้
4. Click **"Full Screen"** to maximize the display.  
   กดปุ่ม **"Full Screen"** เพื่อแสดงผลเต็มหน้าจอ
5. Use the **Color Picker** or **Sample Buttons** to change the display color.  
   ใช้ **ตัวเลือกสี** หรือ **ปุ่มสีตัวอย่าง** เพื่อเปลี่ยนสีตัวเลข
6. Click **"เปลี่ยนเป็นเลขไทย"** to switch numeral styles.  
   กดปุ่ม **"เปลี่ยนเป็นเลขไทย"** เพื่อสลับรูปแบบตัวเลข

## Customization / การปรับแต่ง

You can modify the source code directly to change colors, fonts, or preset times.  
คุณสามารถแก้ไขโค้ดได้โดยตรงเพื่อเปลี่ยนสี ฟอนต์ หรือเวลาของปุ่มทางลัด
